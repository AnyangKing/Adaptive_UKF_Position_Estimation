# 62. 기하다양성별 도약이득 위상탈상관 검증

## 목적

"도약 이득의 지배 변수는 정지/이동이 아니라 관측 창 내 멀티패스 위상 탈상관"이라는 재프레임을
검증한다. δ≈2·d_s·d_r/(c·X)이므로 접선(방위) 운동은 위상을 거의 안 돌리고 radial·vz 운동이 돌린다는
물리에 기반해, 사전등록 가설 4개를 세웠다: H1) 이득∝1/위상스윙(cycles), H2) 등거리 선회(접선
1 m/s, vz=0)는 빨라도 정지급 이득, H3) vz 추가(59 유사) 시 이득 소멸, H4) radial은 저속 유지·고속
소멸. 600 m, 조건 6종×8기하, 20 ping 정착창(후반 10), 도약 정책 61 동결.

## 방법

- 조건: static / radial 0.05·0.2·1.0 m/s / tangential 1.0(vz=0, 등거리 선회) / tangential 1.0+vz 0.08.
- 각 런에서 예측 위상스윙 cycles = f·(δ_max−δ_min)을 image-source로 계산(GT는 평가·설명변수에만).
- fixed 32 kHz vs 30~34 kHz 도약(20개 순환), 동일 라우팅 UKF, paired 이득. Wilcoxon·Spearman.
- 실행 `python run_motion_sweep.py`, 검증 `python test_diagnostic.py`.

## 결과 (2026-07-07)

H1 전체 상관: Spearman ρ=−0.108 (p=0.47) — **기각**.

| 조건 | cycles | fixed | hop | 이득 | 개선비율 | Wilcoxon p |
|---|---:|---:|---:|---:|---:|---:|
| static | 0.00 | 14.64 | 12.01 | +2.64 | 0.75 | 0.074 |
| radial 0.05 | 0.14 | 12.89 | 12.28 | +0.61 | 0.88 | 0.098 |
| radial 0.2 | 0.64 | 14.48 | 14.01 | +0.47 | 0.75 | 0.37 |
| **radial 1.0** | 2.65 | 10.02 | 7.78 | **+2.24** | 0.75 | **0.027** |
| tangential 1.0 | 0.06 | 16.39 | 11.88 | +4.51 | 0.50 | 0.27 |
| tang 1.0+vz | 3.23 | 16.71 | 15.35 | +1.36 | 0.62 | 0.23 |

- **H1 기각**: 총 위상스윙(cycles)은 이득을 설명하지 못한다. H2는 평균으로만 지지(개선비율 0.50,
  ns), H3는 약화만(소멸 아님), H4는 역전(radial 1.0이 유일하게 유의 +2.24, p=0.027).
- **그러나 6개 조건 전부 — 빠른 이동 포함 — 평균 이득이 양수다.** 59(이동 무효)와 달리 20 ping
  정착창 지표에서는 이동 표적에서도 도약이 이득을 낸다.

## 판정

**진단 (H1 프레임 기각 → 더 나은 백색화 가설 도출).** "총 위상스윙" 가설은 데이터로 반증됐지만,
그 반증이 더 정확한 기전 후보를 드러냈다:

> **도약의 본질 = 편향의 백색화(whitening).** 고정 반송파에서 기하 이동이 만드는 위상 드리프트는
> 느리고 ping간 상관된 오차라서, 칼만필터가 이를 잡음으로 평균하지 못하고 **상태(가짜 속도)로
> 흡수**한다. 도약은 편향을 ping마다 독립 점프하는 백색 오차로 바꿔 필터가 잘하는 일(백색잡음
> 평균)을 하게 만든다. 지배 변수는 총 스윙이 아니라 **ping간 편향 상관시간**이다.

이 해석은 (a) 이동 조건들의 양수 이득(상관 드리프트→백색화 이득), (b) 59의 null(10-ping 초기획득
지표는 정착 이득이 안 보임), (c) 60·61의 정지 이득(상관시간 무한대의 극단) 모두와 정합한다. 성립하면
주장이 "정지 전용"에서 **"정착 추적 일반(이동 포함)의 편향 백색화 관측설계"**로 확장돼 중심축(이동체
위치추정)과 직접 연결된다.

다음(사전등록 후보): (i) **백색화 가설 직접 검증** — 고도각 오차/혁신 시퀀스의 lag-1 자기상관을
fixed vs hop으로 비교(hop이 상관을 죽이는지), (ii) 이동 조건 정착창 이득의 대규모 독립 재검(n≥20,
"이동 표적 정착 추적에서도 유효" 확정 — 61 방식). n=8 소표본이므로 62 자체의 조건별 수치는 방향
제시로만 쓴다.

주의: 600 m 단일 거리, 거리당·조건당 8기하. tangential의 큰 평균이득(+4.5)은 소수 기하가 견인
(개선비율 0.50)한 것으로 bimodal — 대규모 재검 필요.
