## 변경

1. **Discussion 배치 문단 확장**: 설계기준 ΔF·δ≳1을 실무 절차로 —
   "수심·트랜시버 깊이·운용거리에서 δ≈2·d_s·d_r/(cX)를 계산하고 ΔF≥1/δ로 대역을 정하라."
   본 기하(d_s~40m, d_r=30m, X=600m)에서 1/δ~0.4kHz → 4kHz 대역이 ~10주기 커버.
   역으로 깊은 물/짧은 게이트-내 지연에서는 필요 대역이 1/δ로 커져 **트랜스듀서 대역 내 백색화
   불가능 시점을 설계자가 판정** 가능. λ/2 동작점 제약과 함께 배치 가능 영역(envelope)을 한정.
   (섹션 참조는 sec:mech 라벨로 — 하드코딩 번호 없음.)
