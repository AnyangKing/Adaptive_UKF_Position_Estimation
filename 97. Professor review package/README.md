# 97. Professor review package

## Purpose

This folder packages the current manuscript state for professor review.

It is meant to answer three questions quickly:

1. Is the paper axis acceptable?
2. Is the novelty framing defensible?
3. What should be prioritized next: submission formatting, real-water validation, or additional simulation?

## Inputs

- Manuscript with references closed:
  - `96. Citation placeholder closure/manuscript_draft_v4_refs.md`
- Figure package:
  - `95. Fig5 PNG and submission packaging/figures/`
- Method/code consistency audit:
  - `93. Method 세부 코드 대조/`
- v3 method-consistent manuscript:
  - `94. Manuscript v3/manuscript_draft_v3.md`

## Outputs

- `professor_review_brief.md`
- `decision_questions.md`
- `current_submission_state.md`
- `review_package_manifest.md`

## Recommended use

Send or show the professor:

1. `professor_review_brief.md`
2. `decision_questions.md`
3. `96. Citation placeholder closure/manuscript_draft_v4_refs.md`
4. `95. Fig5 PNG and submission packaging/figures/`
