# 50. 현실채널 멀티패스구조 신규feature 탐색

## 목적

49번에서 채널을 현실화해도 기존 peak_margin 예측신호가 오히려 약해졌다(ρ -0.45→-0.22). 그러나
richer 채널은 관측 가능한 반사 도착구조를 더 많이 노출하므로, 그 구조에서 뽑은 **새 blind feature**가
계통 고도각 편향을 예측할 여지가 남아 있다. 이 실험은 ③의 저비용 마지막 갈래로, 멀티패스 도착구조
feature가 편향을 기존 신호보다 잘 예측하는지(→novelty 새 근거) 아닌지(→③ 접기)를 판정한다.

## 방법

- 현실채널(2차 반사 + 거친 표면, roughness=0.3), 거리 100/200/400/600 m, 거리당 20개 기하,
  noise seed 4회 평균. GT는 편향 label에만.
- 기준 센서 매치드필터 도착 peak에서 blind 구조 feature 계산(GT 미사용):
  직접-첫반사 상대지연, 직접-둘째반사 상대지연, 첫반사/직접 진폭비, peak 개수, peak 시간 퍼짐.
- 각 feature와 el_bias의 Spearman 상관을 기존 peak_margin과 비교.
- 실행: `python run_newfeatures.py`, 계약검증 `python test_diagnostic.py`.

## 결과 (2026-07-07)

| feature | Spearman ρ (vs el_bias) | p |
|---|---:|---:|
| second_reflection_delay_us | +0.249 | 0.026 |
| **peak_margin (기존 대조)** | **-0.237** | 0.034 |
| peak_time_spread_us | +0.200 | 0.076 |
| first_reflection_delay_us | -0.162 | 0.152 |
| reflection_amp_ratio | +0.074 | 0.515 |
| peak_count | (상수, 무효) | — |

- **새 구조 feature 중 최고(second_reflection_delay, ρ=0.25)도 기존 peak_margin(ρ=0.24)을 의미있게
  넘지 못한다.** 둘 다 약한 상관대(|ρ|≈0.25, 설명력 한 자릿수 %)에 머문다.
- 나머지 새 feature는 무의미하거나 marginal하다. 즉 richer 채널이 노출한 도착구조는 계통 편향에 대한
  강한 blind 예측정보를 담고 있지 않다.
- peak_count는 관측 peak 상한(6)에 항상 걸려 상수라 무효였다(사소한 계산 아티팩트, 결론과 무관).

## 판정

**진단 (③ 저비용 갈래 소진 · no-go 강화).** 49번(모디스트 현실화가 신호 약화)에 이어, 50번은
현실채널 도착구조에서 뽑은 blind 새 feature도 계통 편향의 강한 예측자가 아님을 보였다. 따라서
저비용 ③ 갈래 두 개(채널 현실화 + 새 구조 feature)가 모두 novelty를 되살리지 못했다. 남은 것은
Bellhop급 완전 현실화(연속 ray-bending·거리의존 SSP)뿐인데, 저비용 단계 두 번의 음성 결과를 볼 때
그 투자(설치·검증 부담 큼)의 기대이익은 낮다.

**종합 함의(오히려 ②를 강화):** 계통 고도각 편향은 3경로 평면반사뿐 아니라 2차 반사·거친 표면을
더한 현실적 채널에서도 (a) 여전히 작지만 존재하고(DOA<1°), (b) blind로 강하게 예측·보정되지 않는다.
이는 ②의 "편향은 환원 불가 하한"이라는 결론이 채널 현실화에 **강건**함을 보여, ② 논문의 일반성
주장을 오히려 뒷받침한다.

**다음(사용자 판단 지점):** (a) Bellhop급 고비용 투자로 ③를 끝까지 밀지, (b) ③를 여기서 접고
49·50을 ②의 "현실 채널 강건성" 결과로 흡수해 논문 정리로 갈지. 권고는 (b) — 저비용 두 갈래가
음성이고, 그 음성 자체가 ②를 강화하는 유용한 결과다.

주의: 거리당 20개·roughness 0.3 단일 설정이다. baseline 파이프라인 정합성은 49번에서 확인됐다.
편향은 gated SRP 격자(0.2°)로 하한 추정된다.
