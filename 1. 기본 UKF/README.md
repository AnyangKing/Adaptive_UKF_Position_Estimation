# 1. 기본 UKF

## 목적

센서 바이어스, adaptive R, gating을 추가하기 전에 8센서 TOA 거리·방위각·고도각
관측을 사용하는 6차원 constant-velocity UKF의 무잡음 기준선을 확립한다.

## 기존 프로젝트에서 옮긴 환경

- 출처: `Kalman Filter/22. 시퀸스 500만개/코드_최종.py`
- 음속: 1500 m/s
- 센서: 반지름 0.033 m, 수직 간격 0.079 m의 45도 엇갈린 2단 8센서 배열
- 궤적: 200시점, 시점당 1 m 이동
- 방향 갱신: `0.8 * 이전 방향 + 0.2 * 무작위 단위벡터` 후 정규화
- 시작 거리: 과거 조건과 동일하게 좌표 원점 기준 100 m

원본 프로젝트는 수정하지 않았으며, 단위는 cm에서 SI 단위(m, s, rad)로 바꿨다.

## 현재 설계

- 상태: `[px, py, pz, vx, vy, vz]`
- 관측: `[range(8), azimuth(8), elevation(8)]`
- 초기 위치: Ground Truth가 아니라 첫 번째 센서의 거리·방위각·고도각으로 계산
- 초기 속도: 0 m/s
- 각도 innovation: `[-pi, pi)`로 wrapping
- 각도 예측 평균: 원형 평균
- 공분산: 대칭화, Cholesky 검사, 유한값/양의 준정부호 검사

`range`는 `sound_speed * TOA`이다. 현재 단계는 절대 TOA 동기화가 가능하다는 가정의
무잡음 기준 모델이다.

## 실행

```powershell
python test_basic.py
python run_smoke.py
```

결과는 `results/noiseless_smoke.json`과 `results/noiseless_smoke.npz`에 저장된다.

## 최초 검증 결과 (2026-07-03)

- 테스트: 6개 모두 통과
- 위치 RMSE: 0.006852 m
- 평균 위치오차: 0.005457 m
- 최대 위치오차: 0.016102 m
- 마지막 시점 오차: 0.004667 m

이 값은 난수 시드 `20260703`, 시작 거리 100 m인 단일 무잡음 궤적의 smoke test
결과이며 논문 성능 결과가 아니다. 무잡음임에도 오차가 정확히 0이 아닌 이유는
constant-velocity 운동모델과 방향이 계속 변하는 실제 생성 궤적 사이의 모델 불일치,
그리고 수치 안정성을 위한 측정 공분산 바닥값 때문이다.
