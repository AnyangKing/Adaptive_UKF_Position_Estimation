# 138. 이론-실측 오버레이 그림 (해석모델 직접 검증)

## 목적

137의 해석모델(2-ray glint 편향, 식 18–20)을 **58번 실측 bias(f) 곡선과 직접 겹쳐** 검증하는
그림을 만들어 원고 기전 절에 삽입한다. 핵심 설계: **진동 주기 1/δ를 fitting하지 않고 image-source
기하에서 독립 계산된 값으로 고정** — 진폭·위상·오프셋만 추정. 주기가 맞으면 그것은 모델의 구조적
검증이다.

## 정직성 이슈와 해결 (중요)

첫 버전은 600m idx3(주기 235 Hz)를 썼는데, 250 Hz 캐리어 스텝의 **분해 한계(2×step=500 Hz)** 아래라
오버레이가 aliased — 수치 R²는 높아도 리뷰어가 바로 잡을 그림이었다. **Nyquist를 만족하는 기하만
사용하도록 수정**: 400m idx1(1/δ=748 Hz), 600m idx5(533 Hz). 이 제약은 그림 캡션·본문에 명시
("주기가 500 Hz 한계 아래인 기하는 unaliased로 표시 불가 → pooled 사인fit 통계가 보수적인 이유").

## 판정

**기전 절 완성.** 경험 관찰(58) → 해석 유도(137) → 이론-실측 직접 검증(138)의 3단이 갖춰져,
기전 주장이 "그림+상관"에서 "기하-고정 예측이 데이터와 일치(R²=0.99)"로 격상. Nyquist 제약을
명시해 정직성 유지.
