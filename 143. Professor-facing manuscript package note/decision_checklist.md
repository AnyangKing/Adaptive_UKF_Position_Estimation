# Decision checklist before submission

## Author and institution

- [ ] Actual author names.
- [ ] Author order.
- [ ] Affiliation format.
- [ ] Corresponding author.
- [ ] ORCID, if target journal requires it.

## Journal and format

- [ ] Final target journal.
- [ ] IEEEtran 유지 여부.
- [ ] If not IEEE, convert to target template.
- [ ] Page/word/figure/table limits.
- [ ] Reference style.

## Scientific scope

- [ ] Submit as simulation + mechanism paper.
- [ ] Add tank/lake/sea experiment before submission.
- [ ] Keep moving-target results as boundary only.
- [ ] Add schedule ablation, if reviewer-risk is considered high.

## Data and reproducibility

- [ ] Public repository or private-on-request.
- [ ] Archive DOI, e.g., Zenodo, if needed.
- [ ] Include frozen schedules.
- [ ] Include validation result JSON/CSV.
- [ ] Include figure-generation scripts.

## Back matter

- [ ] Funding statement.
- [ ] Data availability statement.
- [ ] Acknowledgment.
- [ ] Conflict declaration route.
- [ ] Supplementary material route.
