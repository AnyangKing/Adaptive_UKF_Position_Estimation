# 90. Tables draft

## Purpose

This folder turns the v1 manuscript callouts into table-ready material.

The goal is not to make the result look uniformly positive. The paper is stronger if the tables show the exact boundary:

- strong static 600 m improvement,
- mechanism-level whitening in moving cases,
- no reliable moving-target RMSE gain,
- quasi-static support only up to very slow drift under the present protocol,
- unresolved citation placeholders.

## Outputs

- `table1_prior_art_differentiation.md`
- `table2_numerical_results_summary.md`
- `table3_limitations_future_work.md`
- `table_insertion_notes.md`

## Manuscript use

Recommended placement:

- Table 1: end of Introduction or beginning of Related Work.
- Table 2: Results section after static and moving/quasi-static subsections.
- Table 3: Discussion or Future Work.

## Claim boundary

The table set supports the following conservative claim:

> Carrier-agile pinging can whiten a carrier-locked coherent multipath DOA residual in compact shallow-water USBL tracking. It reproducibly improves static long-range localization, but moving-target RMSE improvement is not established under the current protocol.
