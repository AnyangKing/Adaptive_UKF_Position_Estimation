# 88. Manuscript v1 integration blocks

## 목적

85번 manuscript draft v0를 v1로 올리기 위한 실제 삽입/교체 블록을 만든다.

87번은 수식 재료를 정리한 폴더였고, 88번은 그 수식을 원고 문장 흐름에 맞춰 배치한 “본문 교체 블록”이다. 다음 작성자는 여기의 블록을 사용해 `manuscript_draft_v1.md`를 만들 수 있다.

## 포함 파일

- `v1_section_blocks.md`: Section 2, 4, 5, 6, 7 교체용 본문 블록
- `figure_callouts.md`: 본문에 넣을 Figure/Table callout 문장
- `v1_assembly_checklist.md`: v1 조립 순서와 점검표

## 핵심

v1에서는 다음이 v0보다 좋아져야 한다.

1. observation/state/mechanism 수식이 본문에 들어간다.
2. Fig.1~7과 Table 1~2의 위치가 명확해진다.
3. Results section에서 positive와 boundary 결과가 같은 톤으로 정리된다.
4. “carrier agility = full multipath solver”라는 오해를 막는다.
