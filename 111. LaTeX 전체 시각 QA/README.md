# 111. LaTeX 전체 시각 QA

## 목적

루트 `manuscript.tex`로 빌드한 PDF 전체를 페이지별로 렌더해 조판·그림·표·참고문헌·후면부를
육안 점검한다. (Word 경로 107이 soffice 부재로 못 하던 QA를 LaTeX에서 완수.) 이번엔 원고 수정
없이 검증만 — 이 폴더는 QA 기록.

## 방법

`pdftoppm -png -r 78~90`으로 전 7쪽 렌더 후 각 페이지 육안 점검. 그림 PNG 픽셀 해상도도 확인.

## 결과 (2026-07-13)

문서는 109 조판 정리(scriptsize 표) 이후 **7쪽**(이전 8쪽에서 압축). 페이지별 점검:

| 쪽 | 내용 | 판정 |
|---|---|---|
| 1 | 제목·초록·Index Terms·서론(드롭캡)·수식(1)(2)·인용[1]~[4] | 정상 |
| 2 | 수식(3)~(12)·§2.1 구현 파라미터 | 정상(overfull 해소) |
| 3 | 수식(13)~(17)·§IV·V·VI | 정상 |
| 4 | §VI·VII·VIII 본문 | 정상 |
| 5 | Table II(전폭, 13행 scriptsize) | 정상·가독 |
| 6 | Table III + Fig.1~4(2×2) | 정상 |
| 7 | Fig.5·6 + References[1]~[4] + 후면부 placeholder | 정상 |

- **위젯/과부착(widow·orphan)·그림 넘침·표 넘침 없음.** Overfull은 1건(3.3pt, 무시 가능).
- **그림 해상도**: Fig.1 4571×1896, Fig.2~4·6 ~1994×1151, Fig.5 1672×924 px. 컬럼폭(~3.5in)
  배치 시 약 470~570 DPI로 **300 DPI 인쇄 기준 상회** → PNG로 투고 품질 충분.
- References는 본문 인용 4편만 출력됨(Loomis·Beaujean·Nhat·Qian). `refs.bib`의 Delano 1953(glint
  배경, optional)은 본문 미인용이라 자동 제외 — 정상 동작. 저자가 원하면 `\cite`로 포함 가능.

## 판정

**전체 시각 QA 통과 — 원고는 페이지 전반 출판 품질.** 조판·그림·표·서지 어디에도 수정이 필요한
결함 없음. 벡터 그림 교체는 이 환경에 SVG→PDF 변환기(rsvg/inkscape/cairosvg)가 없어 보류하되,
PNG가 이미 인쇄 해상도를 넘으므로 필수 아님(선택 개선).

## AI 완결 가능 작업의 경계

108~111로 **AI가 사람 개입 없이 할 수 있는 원고 마감(이관·조판·재현·QA)은 사실상 완료**됐다.
남은 것은 본질적으로 사용자 확정 사항: 저자·소속·Funding·Data Availability(체크리스트 C),
저널 선택과 그에 따른 template/length 조정(E), 최근접 선행연구 원문 대조 1회(D). 다음 자율
작업은 이 경계 안에서 보조적인 것(예: 보충자료 매니페스트)만 가능.
