# 100. Manuscript formatting prep

## Purpose

This folder turns the v4 reference manuscript into a clean, journal-neutral source package.

The project is scientifically assembled, but it is not yet in a final journal template. Because the target journal has not been fixed by the professor, this step avoids locking the paper into Sensors, Applied Acoustics, IEEE/OES, or another venue too early.

## Inputs

- `96. Citation placeholder closure/manuscript_draft_v4_refs.md`
- `95. Fig5 PNG and submission packaging/figures/`
- `99. Journal neutral manuscript checklist/`

## Outputs

- `manuscript_clean_source.md`: clean source manuscript copied from v4-ref and lightly patched for current status.
- `equation_numbering_plan.md`: equation labels and conversion notes.
- `figure_table_insertion_plan.md`: figure/table placement and template conversion plan.
- `cover_letter_bullets.md`: conservative cover-letter material that does not overclaim novelty.
- `formatting_todo.md`: next concrete formatting tasks before journal submission.
- `clean_source_change_log.md`: changes made relative to folder 96.

## Current decision

Keep the manuscript journal-neutral. The next step should convert this package into a target-specific manuscript only after choosing the target journal or template.

## Safety note

This folder is safe to commit by itself. Local handoff files, study files, and professor-report documents outside numbered folders remain local-only unless explicitly requested.
