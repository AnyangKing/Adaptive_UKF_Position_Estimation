# 70. 논문 그림 1차 생성

## 목적

69번에서 확인한 원천 결과 JSON을 사용해 논문 본체에 바로 들어갈 수 있는 1차 그림 세트를 생성한다. 새 알고리즘 성능을 주장하는 실험이 아니라, 지금까지의 검증 결과를 논문 구조로 조립하는 단계다.

## 생성 그림

| 파일 | 의미 | 원천 |
|---|---|---|
| `fig2_frequency_agile_bias.png/svg` | 반송파 도약이 정지 기하의 코히어런트 DOA 편향을 줄이는지 | 58번 |
| `fig3_static_600m_paired_rmse.png/svg` | 정지 600 m에서 fixed 대비 hop RMSE가 줄어드는지 | 61번 |
| `fig4_moving_whitening_lag1.png/svg` | 이동 표적에서 백색화는 되지만 RMSE 이득은 재현되지 않는 경계 | 63번 |
| `fig7_crlb_floor.png/svg` | 장거리 sub-meter 목표가 왜 비현실적인지 보여주는 하한/바닥 | 45번 |

## 핵심 해석

- 논문의 중심축은 변하지 않는다: `TOA + TDOA + DOA` 관측을 UKF 계열 필터에 넣어 위치를 추정한다.
- 연구가 뻗어나간 방향은 “필터만 바꾸는 것”이 아니라, 수중 멀티패스가 만드는 DOA 계통 편향을 송신 주파수 도약으로 백색화하는 것이다.
- 긍정 결과는 정지/준정지 장거리 표적이다. 600 m 정지 표적에서 fixed 평균 RMSE 13.01 m가 hop 8.87 m로 감소했다.
- 이동 표적에서는 lag-1 잔차 상관이 강하게 줄었지만, RMSE 개선은 통계적으로 재현되지 않았다. 이 경계 자체가 논문의 중요한 방어선이다.
- CRLB/개구 한계 때문에 현재 66 mm급 배열과 현실 잡음 조건에서 600 m sub-meter 성능을 기대하면 안 된다.

## 재현 방법

```powershell
python "70. 논문 그림 1차 생성/generate_core_figures.py"
```

결과물은 `figures/` 아래 PNG/SVG와 `summary_numbers.json`으로 저장된다.
