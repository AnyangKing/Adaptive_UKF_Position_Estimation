# 55. 직접-표면반사 DOA 일관성 진단

## 목적

원래 novelty("물리 경로 일관성")를 반증된 per-ping 신뢰도 추론이 아니라 직접-반사 DOA 중복으로
재조준해 검사한다. 표면반사는 소스의 표면 image에서 와 직접파와 다른 고도각으로 배열에 도착하므로,
(a) 소스 깊이를 직접 DOA와 독립으로 구속할 수 있고, (b) 직접-반사 DOA 불일치가 array-intrinsic
편향(고도각 의존)을 blind로 드러낼 수 있다는 가설을 진단한다(필터 만들기 전 go/no-go).

## 방법

- 채택 관측이 직접파 창만 gating하는 것과 달리, 2번째 도착(반사) 창을 gating한 SRP-PHAT로 반사경로
  DOA를 얻는다. 거리 100/200/400/600 m, 거리당 8개 기하, noise 4회 평균. 현실 채널(2차반사+거친표면).
- (a) 참 고도각을 직접 DOA만 vs 직접+반사 DOA로 회귀한 R² 비교(반사가 독립 정보를 더하는가).
- (b) 직접-반사 고도각 gap과 el_bias의 Spearman(gap이 편향을 blind 예측하는가). GT는 label로만.
- 실행: `python run_consistency.py`, 계약검증 `python test_diagnostic.py`.

## 결과 (2026-07-07)

| 항목 | 값 |
|---|---:|
| 참 고도각 복원 R² — 직접 DOA만 | 0.997 |
| 참 고도각 복원 R² — 직접+반사 DOA | 0.997 (개선 **+0.000**) |
| 반사 el vs 참 el 상관 | +0.991 |
| 직접-반사 gap → el_bias ρ | +0.241 (p=0.18) |

- **반사 DOA는 직접 DOA와 중복이다.** 직접 DOA만으로 참 고도각이 이미 R²=0.997로 결정돼, 반사
  DOA를 더해도 개선이 0이다. 반사파는 소스 거울상에서 오므로 같은 기하를 되풀이할 뿐 독립 정보를
  더하지 않는다.
- **직접-반사 gap은 편향을 blind로 예측하지 못한다**(ρ=0.24, p=0.18). 직접·반사가 같은 array-intrinsic
  편향을 비슷하게 받아, 둘의 불일치가 편향을 드러내지 못한다.

## 판정

**진단 (negative).** 물리 경로 일관성을 직접-표면반사 DOA 중복으로 exploit하려는 형태는 성립하지
않는다: 반사 DOA는 직접과 기하적으로 중복이라 독립 깊이 구속을 못 주고, 직접-반사 gap도 편향을
드러내지 못한다.

**③ 일곱 탐침 종합(49~55):** 채널 현실화(49)·새 feature(50)·굴절(51)·프로파일 관측가능성(52)·
joint UKF(53)·주파수(54)·직접-반사 DOA 일관성(55) — 서로 다른 일곱 개의 물리 레버가 모두 같은
결론을 가리킨다. 계통 편향은 배열 개구 기하에 고유한 하한이고, 멀티패스·반사경로는 직접파와
기하적으로 중복이라 편향을 blind로 exploit할 강한 신호가 이 시뮬레이션 계열에 존재하지 않는다.
일곱 갈래의 일관된 반증은 그 자체로 강한 characterization이며 ②의 핵심 주장을 압도적으로 뒷받침한다.

연구 태도상 "novelty가 원리적으로 불가능"이라고 단정하지 않는다 — 아직 안 떠올린 레버나 실해역
실측 데이터에는 다른 구조가 있을 수 있다. 다만 시뮬레이션 내 물리 레버 탐색은 일곱 번의 일관된
결과로 수확체감이 뚜렷하므로, 다음 판단은 ② 논문 실체화로 수렴하는 것과 실해역 데이터 확보를
저울질하는 것이다(사용자 판단).

주의: 거리당 8개, 반사 DOA는 2번째 도착 창 gating으로 추출했다. 편향은 gated SRP 격자(0.2°)로
하한 추정된다.
