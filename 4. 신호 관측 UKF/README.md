# 4. 신호 관측 UKF

## 목적

`3. 멀티패스 DOA 개선`의 8채널 수신 신호와 GCC-PHAT를 6차원 UKF에 연결한다.
이 폴더에서는 아직 adaptive R이나 gating을 적용하지 않는다.

## 상태와 관측

- 상태: `[px, py, pz, vx, vy, vz]`
- 관측: `[sensor 0 절대거리, sensor 0 기준 TDOA 거리 7개, 배열 방위각, 배열 고도각]`
- 절대거리: sensor 0 matched-filter TOA × 음속
- TDOA: 28개 GCC-PHAT pair delay를 일관된 센서 지연으로 최소제곱 복원
- DOA: 같은 28개 pair delay의 평면파 최소제곱

절대 TOA에는 송수신 시각 동기화가 가능하다는 가정이 필요하다.

## 측정 공분산

고정 기준값은 절대거리 0.03 m, 센서 지연 거리 0.025 m, DOA 2°이다.
센서 0 기준 TDOA는 모두 같은 기준 오차를 공유하므로 7×7 block의 비대각 성분을
`sigma²`로 구성했다. 동일 신호에서 얻는 TDOA와 DOA 사이의 cross covariance는 아직
모델링하지 않았으며 다음 adaptive 단계의 한계로 남긴다.

## 초기화

Ground Truth를 사용하지 않는다. 첫 배열 DOA ray와 sensor 0 절대거리 sphere의 양의
교점으로 위치를 구하고 초기 속도는 0 m/s로 둔다.

## 실행

```powershell
python test_signal_ukf.py
python run_smoke.py
```

30시점 수중 궤적에서 매 시점 직접파·해수면·해저 반사, Doppler, 유색잡음을 포함한
원시 8채널 신호를 새로 생성한다. 이 결과는 연결 검증용이며 논문 성능 결과가 아니다.

## 최초 결과 (2026-07-03)

- 초기 위치오차: 7.306 m
- 전체 위치 RMSE: 3.356 m
- 5시점 이후 RMSE: 3.209 m
- 최대 위치오차: 7.306 m
- 마지막 위치오차: 3.600 m
- 테스트: 4개 통과

이 결과는 파이프라인은 연결됐지만 위치 성능은 아직 만족스럽지 않다는 뜻이다.
관측오차를 분해하면 다음과 같다.

- sensor 0 절대거리: 평균 -0.0159 m, RMSE 0.0196 m
- 7개 TDOA 거리오차 벡터 norm: 평균 0.00624 m, 최대 0.0202 m
- 방위각: 평균 -0.0216°, RMSE 0.6305°
- 고도각: 평균 -1.5137°, RMSE 1.6890°

주요 병목은 거리보다 고도각의 지속적인 음의 멀티패스 편향이다. 100 m에서 1.5°의
각도 편향만으로도 수 m 횡방향 오차가 생길 수 있다. 따라서 다음 단계는 Q/R 숫자만
튜닝하는 것이 아니라 innovation과 GCC/SRP 불일치로 각도 관측을 감쇠하거나 보정하는
adaptive R 및 gating이어야 한다.
