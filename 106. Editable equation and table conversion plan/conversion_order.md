# Recommended conversion order

## Step 1: Prepare source

Start from:

`105. Abstract and back-matter tightening/manuscript_sensors_candidate_tightened.md`

## Step 2: Equations

Convert equations before tables.

Why:

- equation numbering affects cross-references;
- equations are easy to break silently if converted late;
- adaptive-R and phase-model equations are central to reviewer trust.

## Step 3: Tables

Convert Table 1 first, then Table 2, then decide whether Table 3 stays.

Why:

- Table 1 is the novelty-defense table;
- Table 2 is the claim-boundary table;
- Table 3 is optional and can become prose/supplement.

## Step 4: Figures

Insert or package figures after equation/table layout stabilizes.

Use:

- Fig.1 from folder 101;
- Fig.2--6 from folder 95 or folder 104 package.

## Step 5: Back matter

Keep placeholders visible until the human author provides final decisions.

## Step 6: Final checks

- Abstract word count.
- Figure order.
- Table readability.
- Equation editability.
- References style.
- Claim boundary.
- Data/code availability statement.
