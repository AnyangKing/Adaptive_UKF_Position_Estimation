# 104. Final submission asset package dry run

## Purpose

This folder assembles a dry-run submission package from the latest manuscript and figure assets.

This is not a real submission package yet. It is a packaging rehearsal to reveal missing files, missing human metadata, and conversion blockers before choosing the final journal template.

## Package structure

- `manuscript/`
  - latest Sensors-style free-format candidate from folder 103.
- `figures/`
  - Fig. 1 polished schematic from folder 101.
  - Fig. 2--6 package from folder 95.
- `supplement_candidates/`
  - supplement/source-map planning documents.
- `checks/`
  - package completeness and remaining-decision checks.

## Current package verdict

The dry-run package has the core manuscript and all six figure assets in both PNG and SVG formats.

It is still not submission-ready because human metadata and final conversion decisions are missing:

- author information;
- author contributions;
- funding;
- conflicts of interest;
- final data/code availability statement;
- final journal template;
- native table conversion;
- editable equation conversion;
- final reference style.
