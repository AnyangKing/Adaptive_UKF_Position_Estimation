# Go / no-go gates

## Gate A: submit with simulation/signal-level validation

Go if all are true:

- Target journal accepts numerical/theoretical validation with full reproducibility.
- Full-text citation audit confirms related-work distinctions.
- Manuscript clearly avoids moving-target RMSE and field-validation claims.
- Figures/tables are journal-format ready.

No-go if:

- Target journal requires field validation.
- Related-work audit finds a prior paper with the same post-gating coherent DOA-bias whitening claim.
- The novelty has to be reframed beyond current evidence.

## Gate B: start real-water validation

Go if any are true:

- Professor requires it.
- Target journal is Applied Acoustics / IEEE/OES-style and appears to expect experimental validation.
- Reviewer-risk assessment says simulation-only is too weak.

No-go if:

- Equipment/site access is unavailable.
- The manuscript can be submitted to a suitable target without field validation.

## Gate C: additional simulation

Go if:

- It answers a reviewer-obvious question with low cost, such as hop-span ablation or array-aperture sensitivity.

No-go if:

- It reopens the failed moving-target adaptive schedule direction without a strong predictor diagnostic.

## Gate D: journal formatting

Go if:

- Target journal is selected.

No-go if:

- Target is still undecided; use journal-neutral formatting instead.
