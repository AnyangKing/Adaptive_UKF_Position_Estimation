# 30-day execution plan

This plan assumes professor feedback is unavailable during the period.

## Week 1: manuscript stabilization

Deliverables:

- Freeze `manuscript_draft_v4_refs.md` as the working manuscript.
- Create a journal-neutral manuscript checklist.
- Mark all text that is still Markdown-specific and needs template conversion.
- Compress Section 2.1 if needed without losing reproducibility.

Exit condition:

- One clean manuscript file with no placeholder citations and no stale figure manifest.

## Week 2: journal-style conversion prep

Deliverables:

- Draft a Sensors-style structure:
  - Abstract,
  - Introduction,
  - System and Methods,
  - Results,
  - Discussion,
  - Conclusions.
- Prepare figure and table insertion order.
- Draft cover-letter bullets.

Exit condition:

- Target-neutral manuscript can be converted into a journal template without scientific rewriting.

## Week 3: full-text citation audit

Deliverables:

- Retrieve publisher/IEEE/PDF pages for the four major prior works.
- Confirm exact wording of the related-work distinctions.
- Add one or two USBL calibration references if needed.
- Decide whether Delano 1953 is included or kept as optional radar-glint background.

Exit condition:

- Related Work is safe against "frequency hopping already exists" criticism.

## Week 4: submission package or experiment plan fork

If submitting without field validation:

- format manuscript,
- format references,
- produce final figure set,
- run final claim audit.

If field validation is required:

- write real-water validation protocol,
- define equipment, site, target ranges, and success criteria,
- keep manuscript as pre-field draft.
