# 3. 멀티패스 DOA 개선

`2. 신호 기반 수중 채널`을 보존하고 동일 채널을 복제해 DOA 추정기만 비교한다.

## 비교 방법

1. 센서별 송신파형 matched-filter TOA → 평면파 최소제곱
2. 모든 28개 센서쌍의 대역제한 GCC-PHAT → 평면파 최소제곱
3. 모든 센서쌍을 결합한 3차원 broadband SRP-PHAT

SRP-PHAT는 방위각·고도각 2도 전역 격자를 탐색하고 최적점 주변을 0.2도로 다시
탐색한다. 개별 센서 DOA를 만들지 않고 배열 전체에서 하나의 DOA를 출력한다.

## 실행

```powershell
python test_estimators.py
python run_compare.py
python run_monte_carlo.py
```

## 주의

단일 시나리오에서 좋아진 방법을 곧바로 채택하지 않는다. 거리, 수심, SNR, 반사계수,
소스 방향을 바꾼 Monte Carlo 비교가 필요하다. 또한 SRP peak score의 절대값은 센서쌍
수와 FFT 길이에 의존하므로 그 자체를 측정 신뢰도로 쓰면 안 된다.

## 최초 결과 (2026-07-03)

### 고정 시나리오

소스 `[100, 20, -20] m`, SNR 20 dB 조건의 3차원 방향각 오차:

- matched-filter TOA 최소제곱: 16.611 deg
- GCC-PHAT 최소제곱: 4.086 deg
- SRP-PHAT: 1.804 deg

### 30회 소규모 Monte Carlo

거리 100/300/600 m, 소스 수심 8~85 m, SNR 10/20/30 dB, 해수면·해저
반사계수와 방사속도를 무작위화했다.

| 방법 | 평균 | 중앙값 | 90백분위 | 최대 | 10도 초과율 |
|---|---:|---:|---:|---:|---:|
| Matched-filter | 4.792° | 0.196° | 15.288° | 41.847° | 23.3% |
| GCC-PHAT | **1.353°** | 0.941° | **2.422°** | 7.144° | 0% |
| SRP-PHAT | 1.472° | 1.311° | 2.786° | **3.792°** | 0% |

matched-filter 방식은 보통 매우 정확하지만 멀티패스 peak가 겹치는 일부 조건에서 큰
오차를 만든다. GCC-PHAT는 평균과 90백분위가 가장 좋고 계산도 SRP 격자탐색보다
단순하다. SRP-PHAT는 최대 오차가 가장 작다. 따라서 다음 UKF 기준 관측은 GCC-PHAT로
하고, SRP-PHAT는 강건성 비교군 및 GCC/SRP 불일치 기반 quality 지표 후보로 남긴다.

30회는 구현 방향을 정하기 위한 탐색 실험이며 논문용 Monte Carlo 결과가 아니다.
