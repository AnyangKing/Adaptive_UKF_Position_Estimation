## 추가한 해석모델 (§V.B, 식 18–20 신설)

게이트-내 신호를 **2-ray 코히어런트 혼합**으로 모델링:
- 직접파(진폭 1, 참고도각 ε₀) + 게이트-내 표면반사(상대진폭 ρ, 겉보기 고도각 ε₀+Δε, 간섭위상
  φ=2πfδ+θ_r).
- 위상/센트로이드 DOA 추정기의 코히어런트-합 각도를 Δε 1차로 전개 → **고전 2-source(glint) 편향**:
  - 식(18): b(f) = Δε·(ρ²+ρcosφ)/(1+ρ²+2ρcosφ)

두 성질을 유도:
1. **b(f)는 반송파에 대해 주기 1/δ로 진동** → Fig.2의 반송파 의존·58번의 78~92% 진동성분을 예측.
2. **한 위상주기 평균이 0**: 식(19) ⟨b⟩_φ = (1/2π)∫₀^{2π} b dφ = 0.
   - 표준적분 ∫dφ/(1+ρ²+2ρcosφ)=2π/(1−ρ²), ∫cosφ dφ/(...)=−2πρ/(1−ρ²)가 분자에서 상쇄 → 정확히 0.
   - **이것이 "백색화"의 해석적 진술**: 간섭위상을 한 주기 평균하면 코히어런트 DOA 편향이 0으로 감.

**설계기준 유도**: K개 반송파가 대역 ΔF를 덮을 때 평균편향이 0에 접근하는 조건
- 식(20): **ΔF·δ ≳ 1** (한 위상주기 이상 커버).
- 600m: δ~수ms → 1/δ~수백Hz~1kHz, ΔF=4kHz(30–34)가 여러 주기 커버 → 강한 평균(0.053° 붕괴와 정합).
- 근거리: δ가 5ms 게이트 밖 → 진동항 없음 → 도약 무익(100–200m null과 정합).
- 이동: δ=δ(t)라 고정 반송파에서도 φ 회전·부분 자기평균 → **motion self-whitening의 해석적 기원**.
