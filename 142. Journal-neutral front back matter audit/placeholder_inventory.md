# Placeholder inventory

## Remaining placeholders

| 위치 | 현재 상태 | 필요한 결정 |
|---|---|---|
| `\author{...}` | `First A. Author, Second B. Author, Third C. Author` | 실제 저자명 및 순서 |
| first `\thanks{...}` | author names/affiliations/corresponding author to be completed | 교신저자 및 제출 정보 |
| funding `\thanks{...}` | funding source(s) or no-funding statement | 과제/지원 여부 |
| affiliation `\thanks{...}` | affiliation and e-mail placeholder | 소속/이메일 |
| Supplementary/Data Availability | repository URL/DOI/access policy pending | 공개 저장소/보충자료 정책 |
| Acknowledgment | to be completed or removed | 감사문 여부 |

## Removed or neutralized

| 기존 표현 | 처리 |
|---|---|
| Target journal: IEEE Journal of Oceanic Engineering | target journal not finalized로 변경 |
| IEEE JOE uses... | IEEEtran journal class neutral statement로 변경 |
| Manuscript submitted to IEEE Journal of Oceanic Engineering | Manuscript prepared in IEEEtran journal format로 변경 |
| IEEE JOURNAL OF OCEANIC ENGINEERING header | IEEE JOURNAL FORMAT header로 변경 |
| IEEE JOE back matter comment | generic IEEE journal conventions로 변경 |

## Rationale

사용자가 이전에 특정 저널/양식 선택을 아직 확정하지 않겠다고 했으므로, 원고는 특정 저널명에 묶지 않는
편이 안전하다. 단, IEEEtran journal class 자체는 유지한다. 지금까지의 조판과 section numbering이 모두
IEEEtran 기준으로 맞춰져 있기 때문이다.
