# 59. 반송파 도약 ping 필터 검증

## 목적

58번은 고정 기하에서 장거리 계통 고도각 편향의 78~92%가 반송파-진동 성분이라 도약 평균으로
상쇄됨을 보였다. 이 실험은 필터 수준 검증으로, 이동 궤적에서 ping마다 반송파를 순환(30~34 kHz,
10개)시키고 기존 채택 라우팅 UKF를 그대로 돌려 32 kHz 고정 대비 위치 RMSE가 실제로 떨어지는지
paired 비교한다. 필터·R·라우팅 수정 없음(순수 송신 다양성).

## 방법

- 거리 100/200/400/600 m, 거리당 12개 test 궤적(10 ping, 이동 소스). 같은 궤적·seed에서 반송파만
  (a) 전 ping 32 kHz vs (b) ping k→30+4k/9 kHz 순환. 채택 라우팅 UKF 동일 적용.
- 지표: RMSE(3-ping 이후) paired 개선, 부트스트랩 CI95, Wilcoxon, 발산율. GT는 평가에만.
- 실행 `python run_hopping.py`, 검증 `python test_diagnostic.py`.

## 판정

**기각 (이동 궤적 regime에서 필터 이득 없음) — 단 기전 통찰은 강화되고, 진짜 적용처가 드러남.**

58의 관측 수준 상쇄가 필터 RMSE로 이어지지 않은 이유를 규명했다: 간섭 위상 φ=2π·f·δ는 반송파(f)
로도 변하지만 **기하(δ)로도 변한다.** 이동 소스는 ping마다 δ가 변해 **고정 반송파에서도 편향이
이미 자연 진동·부분 평균**되고 있었다(42번 A의 장거리 within-track 잔존율 0.23~0.48과 정확히
정합). 따라서 도약이 더할 탈상관 여지가 작고, 이동 궤적의 잔여 오차는 잡음·방위 성분이 지배한다.
도약의 미세한 관측 분산 증가가 오히려 평균 RMSE를 살짝 해쳤다(600 m 발산 1건 포함).

**이 기전이 가리키는 도약의 진짜 적용처는 정지/준정지 표적이다.** 정지 표적은 δ 고정 → 편향이
noise 반복에도 불변인 상수(42번) → ping을 아무리 쌓아도 남는 floor(57번). 그 경우 **반송파 도약이
위상을 돌릴 유일한 수단**이고, 58의 고정-기하 결과(600 m 편향 0.632°→0.053°, 위치 6.6→0.55 m
상당)가 곧 정지 표적에서의 기대 이득이다. 다음(60번)은 정지 표적 정착 시나리오에서 fixed vs hop을
검증한다. 성공 시 novelty는 "정지/저속 표적 USBL(계류 모니터링·도킹·비콘 캘리브레이션)에서
frequency-agile pinging에 의한 멀티패스 DOA 편향 자기상쇄"로 명확한 적용범위와 함께 성립한다.

주의: 거리당 12궤적 소표본. 도약 시 라우팅·R은 미조정(도약 관측 분산 증가를 R이 모름) — 정지
표적 검증에서 같은 구성 유지해 순수 송신효과만 본다.
